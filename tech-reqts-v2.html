Technology Requirements

CSC 161: Cloud Infrastructure and Security - Fall 2025

Most Critical Requirement: Reliable Internet Connection

A stable internet connection is essential for success in this course. All course activities depend on internet connectivity:

  • • Zoom Classes: Weekly Thursday meetings (student video not required, but viewing instructor video/screen share)
  • • VPN Access: Part 1 labs require connection to college Proxmox infrastructure
  • • Cloud Platforms: Part 2 labs use AWS, Azure, and GCP web consoles

Minimum: 5 Mbps download, 2 Mbps upload

Recommended: 10+ Mbps download for smoother experience

Course Technology Overview

Part 1: College-Hosted Proxmox Labs (Weeks 1-6)

Access college-hosted Proxmox VE instances through secure VPN connection.

  • • Works on ANY device with OpenVPN client and web browser
  • • Smartphones, tablets, Chromebooks, laptops all supported
  • • College provides enterprise-grade server infrastructure
  • • No local virtualization hardware required
  • • 24/7 VPN access to assigned Proxmox instances

Part 2: Cloud Security Labs (Weeks 7-17)

Multi-cloud exploration using college-provided accounts and AWS free tier for CCSK certification preparation.

  • • Google Cloud via college Google accounts
  • • Azure through college MSLE program (Skillable labs)
  • • AWS personal free tier account required
  • • Estimated total cost: $0-15 for AWS activities only
  • • Identity management, security controls, storage encryption
