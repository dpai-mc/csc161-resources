Assignment: Week 2 Lab

Proxmox VE Initial Setup

Objective

The goal of this assignment is to perform the initial configuration of your Proxmox VE instance. This includes managing software repositories, updating the system, and installing essential administrative tools. You will be guided through this process by the AI Tutor, "Alex," to ensure your server is stable, secure, and ready for future labs.

Background

Properly configuring a new server is one of the most critical tasks for any systems administrator. This "Phase I" setup ensures that your Proxmox host is using the correct, stable software sources and is patched against known vulnerabilities. By interacting with the AI Tutor, you will learn the best practices for these initial steps in a guided, real-world scenario.

Instructions

  1. Prerequisites: Ensure you have successfully completed the Week 1 VPN Connectivity lab. You must be connected to the course VPN to access your Proxmox instance.
  2. Log In: Access your Proxmox VE web interface at `https://[Your-IP]:8006` and log in.
  3. Engage the AI Tutor: Start a new session with the AI Tutor for this assignment. The tutor will introduce himself as "Alex."
  4. Follow the Guided Process: Alex will guide you step-by-step through:
    • Configuring Proxmox software repositories (disabling enterprise, enabling no-subscription).
    • Updating the server operating system.
    • Rebooting the server safely after updates.
    • Installing essential command-line tools.
  5. Capture the Handoff Summary: At the end of the session, Alex will generate a "Handoff Summary" that details the final state of your server. This summary is your proof of completion.

Student Deliverables

You will submit one file to the Canvas assignment portal:

  • A single text file (`.txt`) containing the complete, unedited **Handoff Summary** provided by the AI Tutor, Alex, at the conclusion of your lab session.
