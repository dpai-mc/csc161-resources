Hardware Justification

CPU Requirements

Virtualization requires hardware support (VT-x/AMD-V). Course labs will run 3-5 VMs simultaneously including Proxmox host, pfSense firewall, and multiple guest systems.

Memory (RAM)

Each VM requires 2-4 GB RAM. With host OS overhead, 16 GB allows 2-3 VMs, while 32 GB enables full lab scenarios with 5+ concurrent VMs.

Storage

VM images, ISOs, and snapshots consume significant space. SSD dramatically improves VM boot times and overall lab performance.

Network

Cloud labs require reliable internet for VM creation, software downloads, and remote access. Course involves significant file transfers and cloud API interactions.
