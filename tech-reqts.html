Fall 2025 Hardware & Software

Course Technology Overview

Important Notice

This course requires significant computing resources for virtualization labs. Please review all requirements carefully before the semester begins.

Part 1: Virtualization Labs

Hands-on experience with Proxmox VE hypervisor, virtual machine deployment, and network security.

  • • Proxmox VE installation and configuration
  • • Multiple VM deployments
  • • Virtual networking (VLANs, bridges)
  • • pfSense firewall deployment
  • • File server configuration

Part 2: Cloud Security

Multi-cloud exploration with AWS, Azure, and Google Cloud Platform for security analysis and CCSK preparation.

  • • AWS, Azure, GCP free tier accounts
  • • Virtual machine deployments
  • • Identity and Access Management
  • • Network security controls
  • • Cloud storage security

Reduced Hardware Requirements

Part 1 labs use college-hosted Proxmox VE instances accessed via VPN, dramatically reducing local hardware requirements. Students only need a stable internet connection and basic computer. Part 2 cloud labs may incur minimal charges (typically $5-15 total) for advanced features beyond free tiers.

Hardware Justification

Network Connection

Stable internet is critical for VPN access to college PVE instances and cloud platform work. Higher bandwidth improves remote desktop responsiveness and reduces latency.

Display & Interface

Higher resolution displays improve readability of remote desktop sessions and cloud consoles. Multiple browser tabs and terminal windows benefit from larger screens.

VPN Client Support

OpenVPN client required for secure access to college Proxmox infrastructure. All major operating systems support OpenVPN connections.

Local Storage

Minimal local storage needed since virtualization runs remotely. Space required for VPN client, documentation, screenshots, and cloud CLI tools.
