🖥️ VM Creation Workflow

Learn the step-by-step process of creating a new virtual machine:

  1. Navigate to node → Create VM
  2. Configure general settings (VM ID, name)
  3. Select operating system and ISO
  4. Configure virtual hardware (CPU, memory, disk)
  5. Set up network configuration
  6. Review and create

🔄 System Update Process

Practice the safe system update procedure:

  1. Check current system status
  2. Review available updates
  3. Apply updates systematically
  4. Reboot if kernel updates are installed
  5. Verify system after updates

🧠 Knowledge Check

Question 1: Interface Navigation

Where would you go to configure cluster-wide backup jobs in ProxMox VE?

✅ Correct! Datacenter → Backup is where you configure cluster-wide backup jobs that can affect multiple nodes and VMs.
❌ Incorrect. Cluster-wide backup jobs are configured at the Datacenter level, not at individual node or VM levels.
